
🎨 Modern Gallery Landing Page

Projeto de uma landing page moderna desenvolvida com HTML e CSS puro, com foco em layout visual, responsividade e efeitos de interação.

🚀 Preview

Uma página com:
	•	Hero section com imagem de fundo + overlay
	•	Texto em destaque
	•	Botão de call-to-action
	•	Grid de cards com imagens e descrição
	•	Efeitos de hover suaves

🛠️ Tecnologias utilizadas
	•	HTML5
	•	CSS3 (Flexbox)
	•	Google Fonts (Manrope)
	•	Font Awesome

  🎯 Funcionalidades
	•	Layout responsivo com Flexbox
	•	Hero com background image + gradient overlay
	•	Cards organizados em grid flexível
	•	Efeitos de hover (zoom e destaque)
	•	Uso de variáveis CSS para padronização

💡 Aprendizados

Este projeto foi desenvolvido com foco em:
	•	Estruturação de layout com Flexbox
	•	Uso de variáveis CSS (:root)
	•	Organização de estilos
	•	Criação de interfaces modernas

📸 Layout

O design simula uma página de galeria de arte contemporânea, com foco em estética minimalista e tipografia forte.
